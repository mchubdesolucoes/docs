## Autenticação

<Steps>
  <Step title="Gere suas chaves">
    Acesse o portal do cliente
  </Step>
  <Step title="Configure o Checkout">
    Vá para GATEWAY/CHECKOUT > Checkout
  </Step>
  <Step title="Obtenha as credenciais">
    Gere Client ID (ci) e Client Secret (cs)
  </Step>
  <Step title="Use nos cabeçalhos">
    ```http
    ci: seu-client-id
    cs: seu-client-secret
    ```
  </Step>
</Steps>

## Exemplo de Uso

<CodeGroup>
  ```javascript JavaScript
  // Exemplo de requisição PIX Cash-in
  const response = await fetch('https://ws.relaxpay.site/api/v1/gateway/request-qrcode', {
    method: 'POST',
    headers: {
      'ci': 'seu-client-id',
      'cs': 'seu-client-secret'
    },
    body: JSON.stringify({
      amount: 100.00,
      callbackUrl: 'https://seu-webhook.com'
    })
  });
  ```
</CodeGroup>
