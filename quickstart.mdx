# Início Rápido

Siga este guia para começar a usar a RelaxPay em sua aplicação.

## 1. Crie sua conta

Acesse [app.relaxpay.app](https://app.relaxpay.app) e crie sua conta gratuita.

## 2. Obtenha suas chaves de API

Após criar sua conta, você terá acesso às suas chaves de API:

- **Chave de Teste**: Para desenvolvimento
- **Chave de Produção**: Para ambiente real

## 3. Instale o SDK

Escolha o SDK da sua linguagem preferida:

```bash
# Node.js
npm install @relaxpay/sdk

# Python
pip install relaxpay-sdk

# PHP
composer require relaxpay/sdk
```

## 4. Configure o SDK

```javascript
// Node.js
const RelaxPay = require('@relaxpay/sdk');

const client = new RelaxPay({
  apiKey: 'sua_chave_de_api',
  environment: 'sandbox' // ou 'production'
});
```

## 5. Faça seu primeiro pagamento

```javascript
// Criar um pagamento
const payment = await client.payments.create({
  amount: 1000, // R$ 10,00
  description: 'Pagamento de teste',
  paymentMethod: 'credit_card',
  card: {
    number: '4111111111111111',
    holderName: 'João Silva',
    expirationDate: '12/25',
    cvv: '123'
  }
});

console.log(payment);
```
